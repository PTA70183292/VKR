Требуется Python версии 3.10 для совместимости с стабильными версиями библиотек
    Docker >= 24.x
    Docker Compose v2
    Для Windows:
        Требуется WSL2 (wsl --install)
    Для Linux:
        Нативная поддержка Docker, WSL не требуется 
    Git
GPU Nvidia с поддержкой CUDA
    Драйвер Nvidia (рекомендуется >= 535)
    NVIDIA Container Toolkit (nvidia-docker)
    Минимум 8 GB видеопамяти (рекомендуется 12–16 GB)
Поднять контейнеры: docker compose up --build
Запустить http://localhost:8501/

Для обучения модели из другого датасета:
    Загрузить CSV файл
    Обязательные колонки:
        text
        label
    Есть тестовый датасет для проверки: .\Тестовый датасет для обучения\Датасет для проверки обучения.csv"

Для загрузки модели из файла (в нашем случае адаптера), достаточно загрузить папку адаптера в директорию \backend\trained_models. В таком случае, фронтент часть приложения автоматически увидит адаптер и можно будет проводить анализ текста с выбранным адаптером.

Для проведения тетов: docker-compose exec backend pytest tests/ -v --disable-warnings
